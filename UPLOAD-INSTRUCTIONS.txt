5. Drag them onto GitHub.
6. Use commit message: Upload final Iron Oak website

CANSPACE / CPANEL
1. Open File Manager and enter public_html.
2. Back up the current website before replacing it.
3. Upload this ZIP to public_html.
4. Select the ZIP and click Extract.
5. Make sure index.html is directly inside public_html, not inside another folder.
6. Remove the ZIP after extraction.
